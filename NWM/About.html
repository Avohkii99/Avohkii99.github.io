About Hydropower Forecaster

Hydropower Forecaster is a web application designed to provide users with access to US National Water Model (NWM) forecasting tools. The application allows users to retrieve short-range, medium-range, and long-range forecasts for specific reach IDs by clicking on points from a KML layer on the map. For the purposes of this website, these points represent theoretical hydropower stations that the user has placed around San Jose, California. The application displays expected levels of power generation for each of the individual points in both table and graph formats, providing a comprehensive view of the projected power generation.

Key Components

  • Map: The map is built using the Leaflet library, which provides an interactive map interface. The map displays the NHD Major Rivers and Creeks layer from the Hydroshare GeoServer and a KML layer with clickable points representing different reach IDs.
  • Data Retrieval: When a user clicks on a point on the map, the application retrieves the forecast data for the selected reach ID from the NOAA Water API. The data includes short-range, medium-range, and long-range forecasts, which are displayed in a table and a graph.
  • Data Display: The forecast data is displayed in a table and a graph using the Chart.js library. The table shows the date, time, and power (kWh) for each forecast entry, while the graph provides a visual representation of the power generation over time.
  • Forecast Type Selection: Users can select the type of forecast (short-range, medium-range, or long-range) using a dropdown menu at the bottom of the page. A note next to the dropdown menu informs users that currently, only short-range data is available for these locations, but the functionality will be useful as more extended forecasts become available.

Technologies Used

  • HTML: The structure of the web pages is built using HTML.
  • CSS: The styling of the web pages is done using CSS, including custom styles for the map, table, graph, and other elements.
  • JavaScript: The functionality of the web application is implemented using JavaScript, including data retrieval, data display, and user interactions.
  • Leaflet: The interactive map is created using the Leaflet library.
  • Chart.js: The graph displaying the forecast data is created using the Chart.js library.
  • NOAA Water API: The forecast data is retrieved from the NOAA Water API.
