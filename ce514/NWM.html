How does it work?

This page allows users to retrieve US National Water Model short-range forecasts for a specific reach ID. Users can enter a reach ID in the input box and click the "Get Forecast" button. The application will then fetch the forecast data for the specified reach ID, display the values in a table, and draw the forecast in a graph using Chart.js.
